More about this project.

In the field of analog design it is necessary to create programming tools, which aim to facilitate the process of circuit design. In this work, a tool was created which, using genetic algorithms, finds the parameters of the transistors, that the designer must define in order to achieve a design based on the characteristics he desires. This work has been implemented for two designs, a simple OTA as well as for a Cascoded OTA.

Tools Used for this Project:

  • C
  • Java
  • Genetic algorithm
  • Analog design
  • LTspice
  • OrCAD
